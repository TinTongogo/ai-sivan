## 2. 新增的叶子执行器

```java
@Component
class ImageAnalysisLeafExecutor implements LeafExecutor {

    private final ModelRouter router;

    @Override public String supportedType() { return "image_analysis"; }

    @Override
    public Flux<OrchestrationEvent> execute(ForestNode node, ExecutionContext ctx, EventSink sink) {
        // 从 node.metadata 获取图片 URL
        String imageUrl = node.metadata().get("image_url");
        String prompt = node.content();

        // 使用支持 VISION 的模型
        LanguageModel model = router.forTask(TaskProfile.builder()
            .requiresVision(true).build());

        return model.chat(List.of(
            Msg.of(Role.USER, List.of(
                new Content.Text(prompt),
                new Content.Image(imageUrl)
            ))
        ), ModelParams.defaults()).map(event -> {
            if (event instanceof ChatEvent.Chunk chunk) {
                return OrchestrationEvent.stream(chunk.text());
            }
            return OrchestrationEvent.complete(Map.of("type", "analysis_done"));
        });
    }
}
```

### 2.1 多模态叶子执行器错误恢复

多模态生成/分析可能因多种原因失败：模型不支持、配额不足、内容安全过滤等。用户需要区分失败原因并做后续处理。

**错误分类与恢复策略**：

| 错误类型 | 错误码 | 恢复策略 |
|---------|--------|---------|
| 模型不支持 | `MODEL_NOT_SUPPORTED` | 自动降级为文本描述 |
| 配额不足 | `QUOTA_EXCEEDED` | 提示用户检查 API 配额，或切换到备用模型 |
| 内容安全过滤 | `CONTENT_FILTERED` | 提示用户修改输入内容 |
| 网络超时 | `TIMEOUT` | 自动重试 1 次，仍失败则提示网络异常 |
| 未知错误 | `UNKNOWN` | 返回错误详情 + traceId，引导用户反馈 |

```java
class ImageGenLeafExecutor implements LeafExecutor {
    @Override
    public Flux<OrchestrationEvent> execute(TreeNode node, ExecutionContext ctx, EventSink sink) {
        String prompt = node instanceof ContentNode c ? c.content() : "";
        return modelRouter.forTask(taskProfile)
            .flatMap(model -> model.complete(imageGenRequest(prompt)))
            .onErrorResume(ModelNotSupportedException.class, e ->
                // 降级：LLM 生成文本描述替代图片
                textFallback(prompt)
            )
            .onErrorResume(QuotaExceededException.class, e ->
                Flux.just(OrchestrationEvent.error("quota_exceeded", "API 配额不足，请检查设置"))
            )
            .map(result -> OrchestrationEvent.complete(Map.of("type", "image_gen", "result", result)));
    }

    private Flux<ChatResult> textFallback(String prompt) {
        return modelRouter.forTask(simpleChatProfile)
            .flatMap(model -> model.complete(textRequest("描述一下这个场景: " + prompt)));
    }
}
```
