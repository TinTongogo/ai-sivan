# 共享领域事件定义

> 所有领域事件的集中定义。各文档引用此文件，确保签名一致。
> 对应 ARC-01：事件定义集中管理。

---

## 账号与项目事件

```java
/** 新用户注册完成。触发新手引导初始化 + 默认项目创建。 */
record AccountRegistered(
    UUID accountId,
    String shortId,
    String email,
    boolean oauthOnly,
    Instant occurredAt
) {}

/** 账号状态变更。触发 API Gateway 的 JWT 黑名单更新。 */
record AccountStatusChanged(
    UUID accountId,
    String oldStatus,
    String newStatus,
    Instant occurredAt
) {}

/** 项目已归档。触发 UI 刷新列表 + 文件系统只读标记。 */
record ProjectArchived(
    UUID projectId,
    UUID accountId,
    Instant occurredAt
) {}

/** 项目从隔离区恢复。触发 UI 刷新列表。 */
record ProjectRestored(
    UUID projectId,
    UUID accountId,
    Instant occurredAt
) {}

/** 项目被永久删除。触发文件系统清理 + 索引清除。 */
record ProjectDeleted(
    UUID projectId,
    UUID accountId,
    boolean cleanupFiles,
    Instant occurredAt
) {}
```

## 执行事件

```java
/** 任意节点的 status 变更。触发进度重算 + 依赖检查。 */
record NodeStatusChanged(
    UUID nodeId,
    String nodeType,
    NodeStatus oldStatus,
    NodeStatus newStatus,
    UUID forestId,
    UUID accountId,
    Instant occurredAt
) {}

/** GoalTree 根节点完成。触发依赖树启动 + 模板成功率更新 + 用户通知。 */
record GoalTreeCompleted(
    UUID rootNodeId,
    UUID forestId,
    UUID accountId,
    UUID templateId,   // null 表示非模板创建
    boolean success,
    int totalNodes,
    int completedNodes,
    String summary,
    Instant occurredAt
) {}

/** 叶子节点执行时产生异常。触发错误通知。 */
record NodeExecutionFailed(
    UUID nodeId,
    UUID forestId,
    UUID accountId,
    String errorMessage,
    Instant occurredAt
) {}

/** InnerGoal 下所有子节点完成，即里程碑边界。触发阶段切换。 */
record MilestoneCompleted(
    UUID nodeId,
    UUID forestId,
    UUID accountId,
    int completedChildren,
    int totalChildren,
    Instant occurredAt
) {}
```

```java
/** 工具执行完成。触发反馈记录 + 审计日志。 */
record ToolExecutionFinished(
```

```java
/** 记忆已召回。触发 ForestCompressor 优先级调整。 */
record MemoryRecalled(
```
