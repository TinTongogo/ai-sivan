# ADR-046：Embedding 模型版本管理

> **状态**：✅ 已决策
> **日期**：2026-06-06

## 背景

`vector(1536)` 硬编码维度，更换 embedding 模型（如 Cohere 1024 维）后现有向量无法使用，需全量重算。

## 决策

1. 声明 `vector(4096)`（当前最大常见维度），使用前 N 维。
2. 增加 `embedding_model` 列记录生成模型标识。
3. 查询时按 `embedding_model` 过滤，只与同模型向量计算相似度。
4. 模型升级后旧向量逐步异步重算（后台 Job），不阻塞线上服务。

## 理由

vector(4096) 兼容所有当前主流 embedding 模型的维度。模型版本列保证混合模型查询的正确性。
