# ADR-036：禁止 LeafExecutor 向下转型

> **状态**：✅ 已决策
> **日期**：2026-06-06

## 背景

`AgentLeafExecutor` 等执行器将 `TreeNode` 强制转换为具体子类（如 `(SearchKBNode) node`），违反里氏替换和接口隔离。

## 决策

`LeafExecutor.execute()` 的入参 `TreeNode` 不应强制转换为具体子类：
- 通用属性（content、metadata）走 `ContentNode` / `TreeNode` 接口方法
- 特殊属性放入 `metadata` Map
- 如果某属性被多个执行器使用，考虑提升到 `ContentNode` 或新增子接口

## 理由

向下转型是运行时隐患——传入非预期子类时抛出 ClassCastException。接口方法 + metadata 的组合编译期即可检查。
