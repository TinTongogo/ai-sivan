# ADR-031：产品策略决策——不做视觉编辑器、图谱、全量录制

> **状态**：✅ 已决策
> **日期**：2026-06-05
> **作者**：竞品分析 → 系统架构组

---

## 背景

竞品分析对比了 Dify、Coze、Rewind AI、Personal.ai 等功能后发现多项差距：

| 差距 | 竞品 | 决策 |
|------|------|------|
| G5 本能模板可视化编辑器 | Shortcuts, Taskade | ✅ 可选非核心——模板共享已覆盖 |
| G7 全量录制 | Rewind AI | ✅ 重评为商业模式差异——企业市场不接受 |
| G8 知识图谱 | Personal.ai | ✅ 降级为未来方向——v2.0 ROI 不足 |
| G10 可视化工作流编辑器 | Dify, Coze | ✅ 设计决定不做——自动 DAG 已覆盖 |

## 决策

### G5：本能模板可视化编辑器

模板共享 + 用户间评分排名已覆盖大部分需求。可视化编辑器是高级功能，降为低优先级（Phase 4 按需评估）。

### G7：不做全量录制

- 企业市场不接受屏幕/活动全量录制（隐私合规风险）
- MCP 连接器模式是更好选择——只在需要时通过 API 获取数据
- 这是隐私战略选择，非技术短板

### G8：不做知识图谱

- v2.0 阶段构建知识图谱的 ROI 不足以覆盖实现成本
- Flashback 主动推送 + RAG 覆盖了主要的"帮我想起"场景
- 知识图谱降为 Phase 4 后的未来方向

### G10：不做可视化工作流编辑器

- Dify/Coze 的拖拽编辑器是为 AI 应用开发者设计的
- Sivan 的用户不需要手动管理 DAG——自动生成 + 本能模板（高级可选）已覆盖
- 对普通用户来说，拖拽编辑器是负担不是价值
