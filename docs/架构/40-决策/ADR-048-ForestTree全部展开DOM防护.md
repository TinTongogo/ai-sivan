# ADR-048：ForestTree 全部展开的 DOM 防护

> **状态**：✅ 已决策
> **日期**：2026-06-06

## 背景

1000 节点全部展开时 DOM 元素可达 ~5000+，引发重排卡顿。文档提到虚拟滚动但未具体说明展开全部节点的防护措施。

## 决策

| 防护 | 触发条件 |
|------|---------|
| 自动折叠 | 展开节点 > 50 时，折叠前一层级 |
| `content-visibility: auto` | 始终启用 |
| 虚拟滚动降级 | 展开后元素 > 500 |
| 警告提示 | 展开 > 200 节点时提示用户 |

## 理由

展开全部节点是用户主动行为，系统应做保护性限制但不阻止用户操作。
