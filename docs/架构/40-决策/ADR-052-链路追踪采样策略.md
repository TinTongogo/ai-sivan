# ADR-052：链路追踪采样策略

> **状态**：✅ 已决策
> **日期**：2026-06-06

## 背景

`Tracer` 每次执行都创建完整 trace，高并发下可能压垮 Jaeger。无按场景的采样策略。

## 决策

| 场景 | 采样率 |
|------|--------|
| SUMMARY / CONDITIONAL / CONSENSUS | 100%（全量） |
| SEQUENTIAL / HIERARCHICAL | 50% |
| PARALLEL | 10%（按 forestId 采样） |
| 错误/异常 | 100%（全量） |

## 理由

采样率按场景区分，关键路径（LLM 决策、合成阶段）全量记录，高频路径（PARALLEL）采样降低存储开销。
