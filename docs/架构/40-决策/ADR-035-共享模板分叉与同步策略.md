# ADR-035：共享模板分叉与同步策略

> **状态**：✅ 已决策
> **日期**：2026-06-06

## 背景

共享模板被实例化时执行深拷贝到本地账户。此后原始模板的改进（成功率更新、拓扑优化）不会同步到副本，导致模板分叉和知识孤岛。

## 决策

v2.0 接受模板分叉，暂不实现自动同步：
1. 用户对本地副本的定制可能与原始模板冲突，自动同步可能覆盖个性化修改。
2. 模板共享池的排序算法会自然将高成功率模板推向前。
3. 后续通过 `version` 字段实现增量同步（Phase 3 评估）。

## 理由

同步机制在 v2.0 阶段引入过早——模板生态未成熟前，分叉是正常现象而非问题。
