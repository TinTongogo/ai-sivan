## 核心设计原则

| 原则 | 说明 |
|------|------|
| 零 `switch(nodeType)` | 所有类型分派走 Registry（`LeafExecutorRegistry`、`ModeDispatcher`） |
| 零 `if(delivery)` | 输出过滤走 `EventSink` Decorator 链 |
| 禁止 ThreadLocal | `accountId` 在所有方法中显式传递 |
| 接口隔离 | 一个接口一个变化方向（`TreeNode` / `ExecutableNode` / `CompressibleNode` / `ContentNode`） |
| 领域事件驱动 | 跨上下文通信走事件 record，携带 `forestId` + `accountId` |
| 开闭原则 | 新增类型 = 新类 → 实现接口 → 注册。零个已有文件被修改 |
